# yagopinilla.github.io

Personal website hosted on GitHub Pages — [yagopinilla.com](https://yagopinilla.com)

## Structure

```
/
├── index.html   # Main page
├── style.css    # Styles
├── cv.pdf       # CV (add your own)
└── README.md
```

## Setup

1. Add your compiled `cv.pdf` to the root of the repo.
2. Push to the `main` branch — GitHub Pages will serve the site automatically.
3. Optionally configure your custom domain (`yagopinilla.com`) under **Settings → Pages**.
